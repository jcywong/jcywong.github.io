Cf Telegraph图床

Feb 5, 2024

telegraph 图床

https://github.com/ifyour/cf-image-hosting

特点

  • 免费且无限制
  • 拖放上传
  • 复制并粘贴以上传
  • 最大文件大小为 5MB

步骤

  1. git clone 项目到本地
  2. 终端 cd 到项目目录
  3. 安装好 Node.js，确认可用使用 npm
  4. 注册好 Cloudflare 并登录
  5. 执行 npm i && npm run deploy
  6. 会弹出要求 Cloudflare 授权
  7. 成功！然后去 Cloudflare 绑定自己的域名

deploy

$ git clone https://github.com/ifyour/cf-image-hosting.git
$ cd cf-image-hosting
$ npm install && npm run deploy

bind a domain name

Triggers -> Custom Domains -> Add a custom domain.
