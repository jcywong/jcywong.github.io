优化老款Android TV：精简提速指南

Mar 4, 2024

背景

卧室电视是2019年购买的型号为小米L43M5-AD ，内存为1GB。过了5年，系统版本来到了1.3.130，现在打开电视到能操作需要经过约至少2分钟以上才能使用遥控器，导致卧室电视已经很久没有打开了。在过年期间想着可以抽空看看片子，因此有了此篇文章。

【小米L43M5-AD】小米电视4A 43英寸青春版 全高清 蓝牙语音遥控 1GB+8GB 人工智能网络液晶平板电视 L43M5-AD/L43M5-5A【行情 报价 价格 评测】-京东

方案

Android设备优化不外乎换个rom，或者精简系统。奈何现在设备厂商的封闭，第三方rom越来越少，更别说TV的rom了。因此只能精简系统，移除预装应用程序、服务或者功能，以减少系统的负担，提高系统的响应速度和运行效率。

实施

资源准备

  1. 下载ADB工具并解压，Android SDK Platform Tools；

  2. 下载替换桌面APP存放于ADB解压后的目录下，当贝桌面 或者 Emotn UI ;

  3. 下载对应系统的批处理程序存放于ADB解压后的目录下，Windows、MacOS；

环境准备

  1. **开启开发者模式：**设置 -> 关于 -> 产品型号 -> 连续点击多次；

  2. **开启ADB调试：**设置 -> 账号与安全 -> ADB调试 -> 允许；

  3. 查询电视IP地址： 设置 -> 关于 -> 网络信息 -> IPV4地址；

操作

  1. 点开电脑的终端，使用cd命令切换到ADB目录下；

  2. 在终端中输入adb connect 192.168.199.120 ；

  3. 第一次连接会失败，需要在电视上允许调试；

  4. 再次终端中输入adb connect 192.168.199.120；

  5. 安装桌面app，在终端中输入 adb install ./XXX.apk XXX.apk为实际文件名称；

  6. 执行批处理文件进行精简操作，window系统双击mitv.bat（未验证）执行即可，macos需要赋予脚本执行权限chmod +x mitv.sh 然后运行脚本：./mitv.sh （已验证）；

  7. 重启电视（批处理中执行了卸载桌面，必须在重启电视前安装替换的桌面APP，否则重启后会黑屏）。

总结

优点：

  • 启动速度大大提升，操作很流畅；
  • 我使用的Emotn UI，界面很美观，很简洁；
  • 想要恢复很简单，通过设置中的恢复出厂设置即可；

缺点：

  • Emotn UI无法切换输入源；
  • 卸载了应用商店，安装程序需要通过ADB安装。
